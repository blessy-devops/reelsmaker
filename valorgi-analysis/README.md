# Análise Estratégica - Vídeos Valorde

Este projeto contém análises detalhadas de vídeos virais do canal Valorgy Project, com foco em identificar padrões replicáveis para criação de conteúdo em português.

## Começar por Aqui

**Novo no projeto?** Leia nesta ordem:

1. **[INDICE_ANALISES.md](INDICE_ANALISES.md)** - Visão geral de tudo que foi analisado
2. **[RESUMO_EXECUTIVO_VIDEOS_01-07.md](RESUMO_EXECUTIVO_VIDEOS_01-07.md)** - Padrões, templates e insights dos primeiros 7 vídeos
3. **[analises/](analises/)** - Análises individuais detalhadas de cada vídeo

## O Que Você Encontra Aqui

### Análises Individuais (analises/)
Cada análise contém:
- Transcrição completa em inglês
- Categoria/estilo do vídeo
- Estrutura narrativa detalhada
- Outline dissecado (beat por beat)
- Elementos de viralização
- Padrões técnicos (ritmo, tom, pausas)
- Aplicabilidade ao mercado brasileiro
- Template extraído para replicação
- Variações possíveis

### Resumo Executivo
Consolidação estratégica com:
- Padrões identificados nos 7 vídeos
- 6 templates prontos para uso
- Fórmulas de virabilidade
- Top 10 frases mais virais
- Checklist de qualidade
- Plano de ação em 4 fases
- Ideias de conteúdo para criação imediata

## Templates Disponíveis

1. **Metáfora Animal** - Águia vs Corvo
2. **História Real com Contraste** - Joshua Bell
3. **Experimento Contrastante** - Sapo na água
4. **Lista Numerada** - 5 sinais, 4 realidades
5. **Perspectivas Opostas** - Informação assimétrica
6. **Plot Twist Narrativo** - Cookies no aeroporto

## Como Usar

### Para Criadores de Conteúdo
1. Escolha um template do Resumo Executivo
2. Leia a análise detalhada do vídeo correspondente
3. Adapte usando as sugestões brasileiras
4. Use o checklist de qualidade antes de publicar

### Para Estrategistas
1. Analise os padrões de viralização
2. Implemente o plano de 4 fases
3. Acompanhe as métricas sugeridas

### Para Roteiristas
1. Use os outlines dissecados como referência
2. Adapte as estruturas narrativas
3. Incorpore os gatilhos psicológicos identificados
