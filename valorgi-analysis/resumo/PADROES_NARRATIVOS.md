# Padrões Narrativos - Valorde

## Estrutura Padrão dos Vídeos Virais

### 1. HOOK (0-5 segundos)
**Objetivo:** Capturar atenção imediata

**Tipos de Hook usados:**
1. **Pergunta retórica:** "Did you know that...?"
2. **Fato surpreendente:** "In 1983, at the start of..."
3. **Afirmação chocante:** "We're all going to die one day"
4. **Setup de história:** "A woman was waiting at an airport..."
5. **Promessa de lista:** "5 signs that she is wife material"
6. **Personagem famoso:** "Albert Einstein was teaching a class..."

**Padrão de sucesso:**
- Máximo 10 palavras
- Cria curiosidade ou emoção
- Promete valor específico

---

### 2. DESENVOLVIMENTO (5-50 segundos)
**Objetivo:** Entregar valor e manter engajamento

**Estruturas identificadas:**

#### A) Narrativa Linear (Histórias)
```
Setup → Conflito → Desenvolvimento → Clímax → Resolução
```

#### B) Lista Numerada
```
Número 1 → Número 2 → ... → Número N
```

#### C) Problema-Solução
```
Problema apresentado → Contexto → Solução → Resultado
```

#### D) Metáfora Expandida
```
Situação → Simbolismo → Aplicação → Moral
```

---

### 3. CLÍMAX/VIRADA (variável)
**Objetivo:** Momento de maior impacto emocional

**Tipos de clímax:**
1. **Plot Twist:** Revelação que muda tudo (Video 06 - cookies)
2. **Estatística impactante:** "Over 1,500 people died" (Video 34)
3. **Vitória do protagonista:** Cliff Young venceu (Video 28)
4. **Realização filosófica:** "Maybe you are the one who is wrong"
5. **Justiça poética:** Corvo cai por falta de oxigênio (Video 05)

---

### 4. FECHAMENTO/MORAL (últimos 10-15 segundos)
**Objetivo:** Deixar mensagem memorável + CTA

**Estrutura padrão:**
```
Moral/Lição → CTA Valorde
```

**CTA padrão usado:**
> "If you're not following the Valorde Project, you're probably never going to see us again. If you are, then congratulations, you're growing every day."

**Variações:**
- "We have content that helps you become 1% better each day"
- "Follow to learn something new every day"
- "If you want more valuable and useful content, follow Valorde"

## Padrões de Viralização

### Emoções Principais Evocadas

| Emoção | Frequência | Exemplos |
|--------|------------|----------|
| Inspiração | 40% | Cliff Young, George Müller |
| Curiosidade | 30% | Singapura, Whale Fall |
| Reflexão | 20% | Cookies, Mortalidade |
| Admiração | 10% | Messi, Rolls Royce |

### Gatilhos Psicológicos

1. **Identificação:** Espectador se vê na história
2. **Aspiração:** Quero ser como o protagonista
3. **Justiça:** Vilão punido, herói recompensado
4. **Curiosidade:** "Como isso funciona?"
5. **Validação:** "Eu sabia!" ou "Faz sentido!"
6. **FOMO:** Medo de perder conteúdo

### Elementos de Compartilhamento

**Por que as pessoas compartilham:**
1. **Parecer inteligente:** "Olha o que eu descobri"
2. **Ajudar outros:** "Você precisa ver isso"
3. **Expressar valores:** "Isso representa o que acredito"
4. **Entretenimento:** "Muito legal!"
5. **Conexão:** "Lembrei de você"

## Templates Extraídos

### Template 1: Metáfora Animal
```
[Animal inferior] ataca [Animal superior]
[Superior] não revida, faz [ação alternativa]
[Consequência natural] derrota [inferior]
MORAL: "Aprenda com [superior], não lute com [inferior]"
```

### Template 2: História com Plot Twist
```
[Protagonista] em [situação comum]
[Antagonista aparente] faz [ação irritante]
[Escalada do conflito]
[Protagonista] se afasta indignado
[REVELAÇÃO] que inverte tudo
MORAL: "Antes de julgar, pense"
```

### Template 3: Fato Curioso → Lição
```
[Fato surpreendente sobre X]
[Explicação técnica de como funciona]
[Escala/números impressionantes]
MORAL: [Aplicação para vida do espectador]
```

### Template 4: História de Famoso
```
[Nome famoso] em [situação específica]
[Desafio ou contexto]
[Ação surpreendente/admirável]
[Resultado/impacto]
MORAL: [Lição extraída]
```

### Template 5: Lista com Gancho
```
"[Número] [sinais/tipos/regras] de [tema]"
1. [Item + explicação curta]
2. [Item + explicação curta]
...
N. [Item mais impactante por último]
CTA
```

## Padrões Técnicos

### Duração Ideal
- **Sweet spot:** 50-75 segundos
- **Máximo recomendado:** 90 segundos
- **Mínimo viável:** 30 segundos

### Ritmo de Fala
- **Palavras por minuto:** 130-150
- **Pausas estratégicas:** Após revelações importantes
- **Aceleração:** Em listas para manter ritmo
- **Desaceleração:** Na moral/fechamento

### Tom de Voz
- **Grave e autoritativo:** 80% dos vídeos
- **Calmo e sábio:** Para filosofia
- **Narrativo:** Para histórias
- **Entusiasmado:** Para fatos curiosos

### Estrutura de Frases
- **Curtas e diretas**
- **Sem jargão complexo**
- **Vocabulário acessível**
- **Repetição estratégica de conceitos-chave**

## Aplicação para Criação de Conteúdo

### Checklist de Vídeo Viral

- [ ] Hook captura atenção em 3 segundos?
- [ ] Promessa clara do que o espectador vai ganhar?
- [ ] Estrutura narrativa clara (início, meio, fim)?
- [ ] Momento de clímax/impacto emocional?
- [ ] Moral aplicável à vida do espectador?
- [ ] CTA que cria urgência ou valor?
- [ ] Duração entre 50-90 segundos?
- [ ] Tom de voz adequado ao conteúdo?
- [ ] Frases curtas e acessíveis?
- [ ] Elemento compartilhável (citação, fato, história)?

### Fórmula de Hook

```
[Elemento surpreendente] + [Promessa de valor] + [Especificidade]
```

**Exemplos:**
- "A única ave que ousa bicar uma águia é o corvo"
- "Em 100 anos, nada disso vai importar"
- "Esta garota colocou a mão na mesma água que afundou o Titanic"

### Fórmula de Moral

```
[Ação imperativa] + [Razão] + [Benefício implícito]
```

**Exemplos:**
- "Aprenda com a águia, não lute com os corvos"
- "Antes de ficar com raiva, pense - talvez você esteja errado"
- "Olhe para a natureza - as respostas já estão lá"
